# Fondo Viaje Diciembre

Este repositorio contiene el proyecto para gestionar el fondo de viaje de diciembre.

## Contenido

- **index.html**: Archivo principal del proyecto.

## Cómo contribuir

1. Haz un fork del repositorio.
2. Crea una nueva rama para tus cambios: `git checkout -b nombre-rama`.
3. Realiza tus cambios y haz commit: `git commit -m "Descripción de los cambios"`.
4. Sube tus cambios: `git push origin nombre-rama`.
5. Crea un Pull Request en GitHub.
